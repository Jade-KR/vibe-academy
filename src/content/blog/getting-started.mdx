# vibePack에 오신 것을 환영합니다

vibePack은 Next.js 기반의 SaaS 보일러플레이트입니다.

## 주요 기능

- **인증**: 이메일/비밀번호, 소셜 로그인, 매직 링크
- **결제**: Polar 연동 구독 및 단건 결제
- **이메일**: Resend로 트랜잭셔널 이메일 발송
- **모니터링**: Sentry 에러 추적
- **다국어**: 한국어/영어 지원

## 빠른 시작

pnpm install 후 pnpm dev로 개발 서버를 실행하세요.
